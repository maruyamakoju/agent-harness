# Specifications

Product specifications and design documents are stored here.

## File Naming Convention

- `spec-NNN-<topic>.md` — Numbered specifications
- `api-spec.md` — API specification
- `data-model.md` — Data model specification
- `architecture.md` — Architecture overview

## Template

```markdown
# Spec: <Title>
- **ID**: spec-NNN
- **Author**: Agent
- **Date**: YYYY-MM-DD
- **Status**: draft | approved | implemented

## Overview
<high-level description>

## Requirements
<detailed requirements>

## Design
<design details>
```
