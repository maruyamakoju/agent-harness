Settings & System Info

Notification Channels

Configure notification channels via environment variables in your .env file.

TELEGRAM_BOT_TOKEN + TELEGRAM_CHAT_ID - Telegram notifications

DISCORD_WEBHOOK_URL - Discord webhook notifications

WEBHOOK_URL - Generic / Slack-compatible webhook

The agent sends notifications for: job_start, job_done, job_failed, circuit_breaker, stale_heartbeat, auth_failed, quota_exceeded.

Dashboard Authentication

Set DASHBOARD_TOKEN in your .env to enable token-based auth. When set, all routes require authentication via browser cookie or Authorization: Bearer <token> header.

Agent Configuration

MAX_JOBS_PER_DAY - Daily quota limit (default: 20)

JOB_COOLDOWN - Seconds between jobs (default: 60)

DEFAULT_MODEL - Claude model for coding (default: claude-sonnet-4-5-20250929)

POLL_INTERVAL - Job queue poll interval in seconds (default: 30)

AGENT_WATCH_REPOS - GitHub repos to watch for "agent" labeled issues
